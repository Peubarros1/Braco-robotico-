 Braço Robótico Jogando Jogo da Velha

 Escopo
Desenvolver um sistema no qual um braço robótico seja capaz de jogar jogo da velha contra um jogador humano. O sistema deve identificar o estado atual do tabuleiro, calcular a melhor jogada utilizando um algoritmo de decisão e executar o movimento correspondente no tabuleiro por meio do braço robótico.
